# Techping

## Overview
Bot that scrapes inventory information from popular tech retailer websites, and pings user whenever a specified item is in stock or drops in price.

1. canadacomputers.com
2. memoryexpress.com
3. bestbuy.ca
4. newegg.ca
5. amazon.ca
